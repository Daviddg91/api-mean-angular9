WebApp y API stack MEAN

Esta es una aplicación Web realizaba sobre el stack de tecnologías MEAN, consiste en un desarrollo NodeJs sobre el framework Angular que consume una API Express contra una base de datos MongoDB, esta aplicación soporta:

  • Funcionalidades CRUD
  • Login de usuario con JWT
  • Proteccion de rutas
  • Datatables con ordenacion, búsqueda y paginación de registros en una tabla
  • Validaciones contra la API customizadas
  • Tecnologías de front-end como Bootstrap y Jquery
